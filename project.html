AI Student Assistant — MERN + RAG

GitHub ↗ Live ↗ Write-up →

A full-stack AI assistant that answers student questions using uploaded academic documents with a retrieval-augmented generation pipeline.

what it does

  • Users upload PDF/DOCX/TXT files and ask natural language questions.
  • Searches document chunks using semantic similarity, answers with contextual evidence.
  • Maintains conversation history for follow-up questions.

how it is built

  • Frontend in React for chat UI, file upload flow, and session control.
  • Backend in Node.js/Express for orchestration and API endpoints.
  • RAG workflow with LangChain to split, embed, index, and retrieve content.
  • MongoDB for user/session metadata and chat history.
